import { EndpointDetails, Hr, Properties } from '../../components'

# NBA Teams

This endpoint allows you to retrieve all NBA teams, a single NBA team by its id, or a random NBA team. The response will be an array of JSON objects if you request all teams, or a single JSON object if you request a single team or a random team.

<Properties
  options={[
    ['id', 'integer', 'The id of the team.'],
    ['city', 'string', 'The city of the team.'],
    ['conference', 'string', 'The conference of the team.'],
    ['division', 'string', 'The division of the team.'],
    ['full_name', 'string', 'The full name of the team.'],
    ['name', 'string', 'The name of the team.'],
  ]}
/>

<Hr />
### All entries

<EndpointDetails path="/nba-teams" />

This endpoint allows you to retrieve all NBA teams.

```json filename="Response"
[
  {
    "id": 1,
    "city": "Atlanta",
    "conference": "East",
    "division": "Southeast",
    "full_name": "Atlanta Hawks",
    "name": "Hawks"
  },
  {
    "id": 2,
    "city": "Boston",
    "conference": "East",
    "division": "Atlantic",
    "full_name": "Boston Celtics",
    "name": "Celtics"
  }
  // ...
]
```

<Hr />
### Single entry

<EndpointDetails path="/nba-teams/<id>" />

This endpoint allows you to retrieve a single NBA team by its id.

```json filename="Response"
{
  "id": 1,
  "city": "Atlanta",
  "conference": "East",
  "division": "Southeast",
  "full_name": "Atlanta Hawks",
  "name": "Hawks"
}
```

<Hr />
### Random entry

<EndpointDetails path="/nba-teams/random" />

This endpoint allows you to retrieve a random NBA team.

```json filename="Response"
{
  "id": 5,
  "city": "Chicago",
  "conference": "East",
  "division": "Central",
  "full_name": "Chicago Bulls",
  "name": "Bulls"
}
```
