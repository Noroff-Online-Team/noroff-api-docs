import { EndpointDetails, Hr, Properties } from '../../components'

# Jokes

This endpoint allows you to retrieve all jokes, a single joke by its id, or a random joke. The response will be an array of JSON objects if you request all jokes, or a single JSON object if you request a single joke or a random joke.

### The Joke model

<Properties
  options={[
    ['id', 'integer', 'The id of the joke.'],
    ['type', 'string', 'The type of the joke.'],
    ['setup', 'string', 'The setup of the joke.'],
    ['punchline', 'string', 'The punchline of the joke.'],
  ]}
/>

<Hr />
### All entries

<EndpointDetails path="/jokes" />

This endpoint allows you to retrieve all jokes.

```json filename="Response"
[
  {
    "id": 1,
    "type": "general",
    "setup": "What did the fish say when it hit the wall?",
    "punchline": "Dam."
  },
  {
    "id": 2,
    "type": "general",
    "setup": "How do you make a tissue dance?",
    "punchline": "You put a little boogie on it."
  }
  // ...
]
```

<Hr />
### Single entry

<EndpointDetails path="/jokes/<id>" />

This endpoint allows you to retrieve a single joke by its id.

```json filename="Response"
{
  "id": 1,
  "type": "general",
  "setup": "What did the fish say when it hit the wall?",
  "punchline": "Dam."
}
```

<Hr />
### Random entry

<EndpointDetails path="/jokes/random" />

This endpoint allows you to retrieve a random joke.

```json filename="Response"
{
  "id": 120,
  "type": "general",
  "setup": "How do locomotives know where they're going?",
  "punchline": "Lots of training"
}
```
