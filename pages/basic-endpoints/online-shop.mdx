# Online Shop

This endpoint allows you to retrieve all products, or a single product by its id. The response will be an array of JSON objects if you request all products, or a single JSON object if you request a single product.

### The Online Shop model

<Properties
  options={[
    ['id', 'integer', 'The id of the product.'],
    ['title', 'string', 'The title of the product.'],
    ['description', 'string', 'The description of the product.'],
    ['price', 'float', 'The price of the product.'],
    ['discountedPrice', 'float', 'The discounted price of the product.'],
    ['imageUrl', 'string', 'The image url of the product.'],
    ['rating', 'integer', 'The rating of the product.'],
    ['tags', 'array', 'A string array of the tags of the product.'],
    [
      'reviews',
      'array',
      'An array of product reviews. Each review has an "id", "username", "rating", and "description".',
    ],
  ]}
/>
