# Rainy Days

This endpoint allows you to retrieve all old games, a single old game by its id, or a random old game. The response will be an array of JSON objects if you request all games, or a single JSON object if you request a single game or a random game.

### The Rainy Days model

<Properties
  options={[
    ['id', 'string', 'The id of the product.'],
    ['title', 'string', 'The title of the product.'],
    ['description', 'string', 'The description of the product.'],
    ['gender', 'string', 'Whether the product is made for male or female.'],
    ['sizes', 'array', 'A string array of the sizes of the product.'],
    ['baseColor', 'string', 'The main color of the product.'],
    ['price', 'float', 'The price of the product.'],
    ['discountedPrice', 'float', 'The discounted price of the product'],
    ['onSale', 'boolean', 'Whether the product is on sale or not.'],
    ['image', 'string', 'The image of the product.'],
    ['tags', 'array', 'A string array of the tags of the product.'],
    ['favorite', 'boolean', 'Whether the product is a favorite or not.'],
  ]}
/>
